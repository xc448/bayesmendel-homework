Statistics warm-up

Consider a random variable \(X\) with a probability density function

\[ f(x) = \frac{c}{\sqrt{x(b-x)}}, \quad 0<x<b, \] where \(c\) is a normalising constant and \(b\) is a parameter.

  • Find \(c\) such that the probability density function is valid. \[ \begin{eqnarray}\\ \int_{0}^{b}\frac{c}{\sqrt{x(b-x)}} dt &=& 1 \\ \Rightarrow -2\sin^{-1}(\sqrt{1-\frac{x}{b}})\Biggr|_{0}^{b} &=& \frac{1}{c}\\ \Rightarrow 0 - (-\pi) &=& \frac{1}{c}\\ \Rightarrow c &=& \frac{1}{\pi} \end{eqnarray} \]

  • Find the corresponding cumulative distribution function, \(F(x)\) and its inverse \(F^{-1}(x)\). \[ \begin{align} F(x) &= \int_{0}^{x} \frac{1}{\pi\sqrt{t(b-t)}} dt\\ &= -\frac{2}{\pi}\sin^{-1}(\sqrt{1-\frac{t}{b}})\Biggr|_{0}^{x}\\ &= -\frac{2}{\pi}\sin^{-1}(\sqrt{1-\frac{x}{b}}) + 1\\ \end{align} \]

In order to get the inverse function \(F^{-1}(x)\), represent \(F(x)\) by y, switch y and x, then solve for y.

\[ \begin{eqnarray} x &=& -\frac{2}{\pi}\sin^{-1}(\sqrt{1-\frac{y}{b}}) + 1 \\ \Rightarrow \sin^{-1}(\sqrt{1-\frac{y}{b}}) &=& -\frac{\pi}{2}(x-1) \\ \Rightarrow \sqrt{1-\frac{y}{b}} &=& \sin(-\frac{\pi}{2}(x-1)) \\ \Rightarrow 1-\frac{y}{b} &=& \sin^2(-\frac{\pi}{2}(x-1)) \\ \Rightarrow \frac{y}{b} &=& \cos^2(-\frac{\pi}{2}(x-1)) \\ \Rightarrow y &=& b\cos^2(-\frac{\pi}{2}(x-1)) \\ \Rightarrow F^{-1}(x) &=& b\cos^2(-\frac{\pi}{2}(x-1)) \end{eqnarray} \]

  • Describe a procedure to generate samples distributed according to \(X\), given a set of uniformly distributed samples \(u_1,\dots,u_n \sim \mathcal{U}[0,1]\).

Generate samples uniformaly distributed at [0,1] and store them in the vector. View this as the F(x). Then plug the vector into the reverse function of cumulative density function to get x distributed according to \(X\).

  • Set the seed to 42 and generate 1,000 samples distributed according to \(X\) using 1,000 uniformly distributed i.i.d. \(\mathcal{U}[0,1]\) samples, with \(b=5\). Plot a histogram of the resulting samples.
set.seed(42)
#generate the samples uniformly distributed at [0,1]
samples <- runif(1000)
#plug into the reverse function of cumulative density function F^{-1}(x)
density <- 5 * (cos(-pi / 2 * (samples - 1)))^2
#plot the histogram
hist(density, breaks = 50)

  • Assume that we have a set of samples \(X_1,\dots,X_n\) which are i.i.d. as \(X\) with \(b\) unknown. Find the log-likelihood function for the parameter \(b\), \(\ell(b;x_1,\dots,x_n)\) and find its derivative with respect to \(b\), \(\frac{d\ell}{db}\). \[ \begin{align} L(b) &= \prod_{i=1}^{n}\frac{1}{\pi\sqrt{x_n(b-x_i)}}\mathbb{1}(0<x_i<b)\\ &= \pi^{-n}(\prod_{i=1}^{n}x_i)^{-\frac{1}{2}}(\prod_{i=1}^{n}(b-x_i))^{-\frac{1}{2}}\mathbb{1}(x_{(1)}>0)\mathbb{1}(x_{(n)}<b) \end{align} \] \[ \begin{align} \ell(b) &= -n\log(\pi)-\frac{1}{2}\sum_{i=1}^{n}\log(x_i)-\frac{1}{2} \sum_{i=1}^{n}\log(b-x_i)+\log(\mathbb{1}(x_{(1)}>0))+\log(\mathbb{1}(x_{(n)}<b))\\ \\ \frac{d\ell}{db}&=-\frac{1}{2}\sum_{i=1}^{n}\frac{1}{b-x_i}+\frac{d}{db}\log(\mathbb{1}(x_{(n)}<b))\\ \end{align} \] We can not technically get the derivative of the indicator function \(\mathbb{1}(x_{(n)}<b)\) since it has a jump discontinuity at \(x_{(n)}=b\).

  • Based on the likelihood, or otherwise, give a statistic based on \(X_1,\dots,X_n\) which is a reasonable estimator for the unknown parameter \(b\). Explain your choice.

In order to maximize \(L(b)\), we need to maximize the term \((\prod_{i=1}^{n}(b-x_i))^{-\frac{1}{2}}\) given that are other terms are fixed for a particular sample.

This term gets maximized when b gets its smallest possible value.

Given that \(b>x_{(n)}\), a reasonable estimator for b based on maximizing likelihood function is \(x_{(n)}\).

Family pedigrees

Read in the .rdata file pedigree.rda. Each data.frame represents a family. Each individual is uniquely identified by the first column called ID. Their sex is coded as 0 for females and 1 for males. Individuals mother and father are indicated in the MotherID and FatherID columns. A value of NA in these columns means that this person is a so-called founder or that a certain parent is missing.

Each pedigree can be thought of as a family tree. For example, a visualisation of a sample pedigree is shown below. The colours indicate affliction status for cancers as labelled in the legend.

In the following exercises, you are encouraged to modularise and comment on your code.

load("pedigree.rda")
  • Write an R function(s) to count the number of unique nuclear families there are in a certain pedigree. A nuclear family is defined as the set of two parents and all of their children.
NuclearFamily <- function(data){
  #select MotherID and Father ID columns that are both not NA
  BothParents <- subset(data, (!is.na(data["MotherID"])) & (!is.na(data["FatherID"])), select = c(MotherID, FatherID))
  #keep only unique rows and count the number of rows, which should be the number of unique nuclear family
  nrow(unique(BothParents))
}
